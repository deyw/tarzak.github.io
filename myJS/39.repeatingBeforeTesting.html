Browser Object Model consists of:

  1. window - is a main object, consists of: other objects of browser, links to other windows, preferences of the window, simple input/output, open/close/print of the windows, location of the window and timers.
  2. navigator - information object
  3. history - buttons forward, backward
  4. location - adress of the web-page
  5. screen - information object about a screen of a user
  6. document
  7. event - list of parameters linked with events, only IE
